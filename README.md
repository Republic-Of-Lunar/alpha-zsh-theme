### Install
This Guide is just for 'oh-my-zsh'
> git clone https://github.com/Memory-Studio/alpha-zsh-theme.git<br/>
> cd alpha-zsh-theme<br/>
> sudo cp -r alpha.zsh-theme ~/.oh-my-zsh/custom/themes<br/>
> vi ~/.zshrc<br/>
> ZSH-THEME="alpha"
